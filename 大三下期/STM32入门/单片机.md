# 单片机

> 2026.6.25

---

## 什么是单片机

- 全称 **Microcontroller Unit**，简称 **MCU**
- 将 **CPU、存储器（RAM/ROM/Flash）、I/O 接口、外设功能模块**集成在一块芯片上的微型计算机系统

---

## 基本术语

| 术语 | 全称 | 说明 |
|------|------|------|
| CPU | Central Processing Unit | 中央处理器，负责执行指令和运算 |
| RAM | Random Access Memory | 随机存取存储器（内存），断电后数据丢失 |
| ROM | Read-Only Memory | 只读存储器，断电后数据不丢失（现代 MCU 多用 Flash 代替） |
| Flash | — | 闪存，可电擦写，断电不丢失，用于存储程序代码 |
| I/O | Input / Output | 输入输出接口，连接外部设备 |

---

## 基本组成

### 核心
- **CPU**：运算和控制中心
- **存储器**：Flash（存程序）+ RAM（存变量/堆栈）

### 主要外设（学习重点）

| 类别 | 常见外设 | 用途 |
|------|----------|------|
| I/O | GPIO | 通用输入输出，控制 LED、按键等 |
| 串行通信 | UART、I2C、SPI | 与传感器、模块、其他芯片通信 |
| 定时器 | TIM、PWM | 精确定时、产生 PWM 波形（电机控制、呼吸灯） |
| 中断系统 | NVIC、EXTI | 对外部事件快速响应，不占用 CPU 轮询 |
| 模拟接口 | ADC、DAC | ADC：模拟量→数字量（采集电压/传感器数据）；DAC：数字量→模拟量 |
| 通信总线 | CAN、USB、Ethernet | 工业/高速/网络通信 |

---

## 单片机的特点

- **高度集成**：一块芯片 = 一台微型计算机
- **低功耗**：适合电池供电的嵌入式设备
- **成本低廉**：适合大批量产品
- **易于开发调试**：IDE 成熟（Keil、IAR、STM32CubeIDE），支持在线调试
- **实时性强**：对外部事件响应快且确定（微秒级），适合工业控制等实时场景
