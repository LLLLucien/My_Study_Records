# pthread 线程

## 线程的生命周期

- 创建 pthread_create()
- 就绪 os调度,获得CPU时间片
- 运行 1.结束2.就绪3.阻塞
- 阻塞 原因：sleep，wait，让出CPU时间片
- 结束

## 分离态与非分离态

- join：非分离态
- detach：分离态

```c
int main()
{
    //创建子线程
    pthread_create(t1);
    //其他语句
    code 1
    pthread_join(t1, NULL); //等待子线程结束，并获得t1返回值
    //如果join没有执行完，后面的语句不会执行
    code 2
    code 3

    //pthread_exit(NULL)
    //结束当前线程，现在就是主线程，进程还在，子线程可以继续执行
    //但是主线程后面的代码就不执行了
    //可以设置返回值，需要void *
    //想要接收，需要用到pthread_join()
    pthread_exit(NULL);

    return 0;
}

void *task(void *arg)
{
    pthread_t tid = pthread_self();

    printf("sub=%lu\n", tid);

 //注意如果函数要返回值都必须动态内存分配
    int *val = malloc(sizeof(int));
    *val = 5;
    pthread_exit((void *)val);
    return NULL;
}
```

### 如何创建分离态线程

```c
int main(int argc, char const *argv[])
{
    // 线程编号
    pthread_t tid;

    // 线程属性

    // 方案一
    // 创建之前使用pthread_attr_init初始化
    // 使用pthread_attr_setdetachstate设置分离态
    // 最后使用pthread_attr_destroy销毁
    pthread_attr_t *attr;
    pthread_attr_init(attr);
    pthread_attr_setdetachstate(attr, PTHREAD_CREATE_DETACHED);

    if (pthread_create(&tid, attr, NULL, NULL) != 0)
    {
        perror("create error\n");
        return 1;
    }

    pthread_attr_destroy(attr);

    // 方案二
    pthread_create(&tid, NULL, task, NULL);
    pthread_detach(tid);
    return 0;
}
```
