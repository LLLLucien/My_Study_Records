# thread 线程

## 进程与线程
进程：进程是程序的一次执行过程，以及在这个过程中获得的CPU、内存、文件网络等资源。
 - 一个程序运行默认一个进程，但可以创建多个子进程
 - 每个进程的资源是独立的，互不干扰，每个进程都有自己的地址空间、文件描述符、信号量等
线程：是进程中的执行单元，一个进程中默认存在一个主线程
- 可以创建多个子线程，多个线程可以共享进程的资源
- 线程的执行是并发的，每个线程可以独立执行，互不干扰

## POSIX

POSIX P可移植 OS-操作系统 I-interface接口，一套技术规范由IEEE定义。
很多操作系统遵循POSIX规范：例如各种UNIX（IBM，Oracle，HP等），Linux，macOS，鸿蒙等
而Windows操作系统不遵循POSIX规范，它有自己的线程模型，称为Win32线程

## Pthread

- POSIX thread
- C++ 11 / boost thread
- OpenMP
- 微软

相关 API
- pthread_t  // 线程ID
- pthread_create()  // 创建线程
    - 返回值：0 成功，-1 失败
```c
     int pthread_create(pthread_t *thread, // 线程ID
                   const pthread_attr_t *attr,// 线程属性
                   void *(*start_routine)(void *),// 线程入口函数
                   void *arg);// 线程入口函数参数
```
- pthread_join()  // 主线程等待join的线程结束
- pthread_self()  // 获取当前线程ID
- pthread_join()  // 主线程等待join的线程结束
- pthread_self()  // 获取当前线程ID
- pthread_exit()  // 线程退出
- pthread_detach()  // 线程分离

- sleep()  // 线程休眠，单位秒
    - 需要头文件 <unistd.h>
    - nanosleep()// 线程休眠，单位纳秒  
- pthread_attr_init()
- pthread_attr_destroy()
- pthread_mutex_init()
- pthread_mutex_destroy()
- pthread_cond_init()
- pthread_cond_destroy()
