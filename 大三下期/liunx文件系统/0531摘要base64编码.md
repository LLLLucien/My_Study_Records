# 文件信息摘要与 base64 编码

## openssl 库
- 安装库：apt install openssl-dev
- 推荐安装文档： apt install openssl-doc

- 一般安装在了usr/include/
- 查看版本：openssl version
- 查看帮助：openssl help

## 摘要算法 MD5,SHA1,SHA256
- 任意长度的输入，输出固定长度的摘要
- 单向、不可逆
- 唯一性（不同的输入，不同的输出）

## base64 编码、解码
- 把任意格式的二进制数据转换成可打印的文本，变大为原来的1.33倍（4/3）
- 可逆，可以解码
### 构成
- 字符 26*2=52 个
- 数字 10 个
- 2个符号 + / =等等
