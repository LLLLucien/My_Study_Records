# io - FILE 操作
## 文件概念

### 广义 ls -l

- 文件      - -：文本文件、二进制文件
- 目录      - d 
- 链接      - l
- 块设备   - block
- 字符设备  - char
- 管道      - p
- 套接字    - s

### 流 stream
- 数据（字节）的序列

- 输入流 in 读取、接收
- 输出流 out 写入、发送

## C API
- 'fopen'   打开文件，获取文件指针
    - 模式：r、w、a、r+、w+、 a+、rb、wb、ab
- 'fclose'  关闭文件，释放文件指针
- 'fgetc' char 从输入流读取一个字符
- 'fputc' char 向输出流写入一个字符
- 'fgets' string 从输入流读取一个字符串，直到遇到换行符
- 'fputs' string 向输出流写入一个字符串
- 'fread' 二进制文件读取多个字节，参数：读取的字节数、读取的字节数的指针、文件指针
- 'fwrite' 向二进制文件写入多个字节，参数：写入的字节数、写入的字节数的指针、文件指针
- 'fseek' 移动文件指针到指定位置
- 'ftell' 获取当前文件指针位置
- 'rewind' 移动文件指针到文件开头
- 'fprintf' 向输出流格式化输出数据
