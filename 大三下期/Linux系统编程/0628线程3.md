## 信号量

- 原子性的整数，进行PV操作

- sem_init(&sem, 0, 1)
- **参数**：
  - 1.信号量本身地址，
  - 2.是否是线程间还是进程间，0表示线程间（一个进程内），1表示进程间
  - 3.1表示初始值为1,可以设置为其他值，例如60
    - 如果是1就称为*二元信号量* ，0或1
    - 如果是其他值就称为计数信号量，可以是任意正整数

P：获得资源 sem_wait(&sem); - sem>0,获得资源,sem-- - sem=0,进入阻塞状态

V：释放资源 sem_post(&sem); - sem++

## 自旋锁 spin_lock

- 与互斥锁都是排他锁，资源加锁后只能被一个线程占用，其他线程只能等待。
- 不适合单核场景，为**多核**场景设计的锁，不断轮询poll以获得锁，避免上下文切换时间，cpu占用率高
  - 适用**临界区小**，临界区大，线程切换时间大，cpu占用时间少的场景
- 自旋锁**不能休眠**，否则会死锁

- 互斥锁：
  - 单核，多核都可以，未获得锁会阻塞，cpu占用低，适用临界区大的场景

- 死锁：多个线程互相持有对方的锁，导致无法继续执行

## 读写锁

- 读共享，写排他

```c
#define _POSIX_C_SOURCE 200809L

#include <pthread.h>
#include <stdio.h>
#include <stdlib.h>
#include <unistd.h>

int n = 0;
void *rd(void *arg)
{
    while (1)
    {
        pthread_rwlock_rdlock(&lock);
        printf("%ld 读取：%d\n", pthread_self(), n);
        sleep(1);
        pthread_rwlock_unlock(&lock);
    }
    return NULL;
}
void *wr(void *arg)
{
    while (1)
    {
        pthread_rwlock_wrlock(&lock);
        n++;
        printf("====写线程，写入:%d\n", n);
        pthread_rwlock_unlock(&lock);
        sleep(3);
    }
    return NULL;
}

int main()
{

    pthread_rwlockattr_t attr;
    pthread_rwlockattr_init(&attr);
    pthread_rwlockattr_setkind_np(&attr, PTHREAD_RWLOCK_PREFER_WRITER_NONRECURSIVE_NP);

    // 两个参数，第一个是读写锁，第二个是初始化参数，可用是NULL
    // 而用这个属性attr可用避免写线程进入饥饿状态
    pthread_rwlock_init(&lock, &attr);

    pthread_t t1, t2, t3, t4;
    pthread_create(&t1, NULL, rd, NULL);
    pthread_create(&t2, NULL, rd, NULL);
    pthread_create(&t3, NULL, rd, NULL);
    pthread_create(&t4, NULL, wr, NULL);

    pthread_join(t4, NULL);
    pthread_join(t1, NULL);
    pthread_join(t2, NULL);
    pthread_join(t3, NULL);

    pthread_rwlock_destroy(&lock);
    pthread_rwlockattr_destroy(&attr);
    return 0;
}

```
