# 文件操作-目录

## API

- opendir()
- readdir()
- closedir()
- mkdir()
- rmdir()

## 示例

```c

    #include <sys/types.h>
    #include <dirent.h>

    int main()
    {
        DIR *dp;
        struct dirent *dp;
        dp = opendir("./");
        if(dp == NULL)
        {
            perror("opendir");
            return -1;
        }
        while((dp = readdir(dp)) != NULL)
        {
            printf("%s\n", dp->d_name);
        }
        closedir(dp);
        return 0;
    }
```
