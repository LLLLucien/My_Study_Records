## 信号量

> 信号与信号量是两种不同的东西

- 信号：进程之间通信的机制
- 信号量：原子性的计数器，线程安全;可以使用`信号量`替代`互斥锁`

### 类型

- 二元信号量：0 或 1
- 计数信号量：任意正整数

### 操作

- P：+
- V：-

### 使用

```c
    sem_init(&sem, 0, 1);
        - 参数：
        - 1.信号量本身地址，
        - 2.是否是线程间还是进程间，0表示线程间（一个进程内），1表示进程间
        - 3.1表示初始值为1
```

```c
    //初始化信号量
    sem_init(&sem, 0, 1);

    // 相对于可以同时跑9个
    sem_init(&sem_sum, 0, 9);

    void *task(void *arg)
    {
        for (int i = 0; i < 100000; i++)
        {
            // 信号值-1
            sem_wait(&sem);
            n++;
            // 信号值+1
            sem_post(&sem);
        }
        return NULL;
    }

    int main(int argc, char const *argv[])
{
    // 1.信号量
    // 2.线程间还是进程间： 0 一个进程内（线程间）；1 多个进程间
    // 3.信号量的值：1 二元信号量；其他值称为计数器信号量
    sem_init(&sem, 0, 1);
    // sem_init(&sem_num, 0, 9);

    pthread_t t1;
    pthread_t t2;
    pthread_create(&t1, NULL, task, NULL);
    pthread_create(&t2, NULL, task, NULL);
    pthread_join(t1, NULL);
    pthread_join(t2, NULL);
    sem_destroy(&sem);
    printf("%d\n", n);
    return 0;
}
```
