## 信号量

> 信号与信号量是两种不同的东西

- 信号：进程之间通信的机制
- 信号量：原子性的计数器，线程安全;可以使用`信号量`替代`互斥锁`

### 类型

- 二元信号量：0 或 1
- 计数信号量：任意正整数

### 操作

- P：+
- V：-

### 使用

```
    sem_init(&sem, 0, 1);
        - 参数：
        - 1.信号量本身地址，
        - 2.是否是线程间还是进程间，0表示线程间（一个进程内），1表示进程间
        - 3.1表示初始值为1
```

```c
    //初始化信号量
    sem_init(&sem, 0, 1);

    // 相对于可以同时跑9个
    sem_init(&sem_sum, 0, 9);
    
    void *task(void *arg)
    {
        for (int i = 0; i < 100000; i++)
        {
            // 信号值-1
            sem_wait(&sem);
            n++;
            // 信号值+1
            sem_post(&sem);
        }
        return NULL;
    }
```
