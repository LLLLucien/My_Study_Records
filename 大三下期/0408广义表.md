# 广义表

## 定义

- 是线性表的一个扩展
- 长度：最外层元素个数
- 深度： 空表为1，看最后括号的重数，例如“)))”
- 表头：第一个元素，空表没有表头和表尾
- 表尾：除了第一个元素，剩下的是表尾

## 存储结构

```c
    typedef struct Inode
    {
        int tag;// 0表示是原子，1表示子表
        union
        {
            ElemType data;
            struct Inode *sublist;
        }val;
        struct Inode *link;
    }
```

# 串

## 定义

- 由0个或多个字符组成的有限序列
- 1+2/n(n+1)

## 串的模式匹配

### Brute-Force算法（暴力算法）

- 目标串：n
- 模式串：m
- 最好：m
- 最差：m*（n-m+1）=>化简 m*n

- 成功：在目标串中找到模式串t，t是s的子串，返回t在s中的下标
- 不成功：返回-1

### KMP算法

- 采用空间换时间
- 采用了next数组，next[j]
  - 模式串相同前后缀的最大长度
  - 例如：
    - ababc
    - 前缀：a,ab,aba,abab(n-1个)
    - 后缀：c,bc,abc,babc

    - 原       a b a b c
    - next数组 0 0 1 2 0 next数组 j=next[j]
    - next数组 -1 0 0 1 2 -1开头的next数组 j=next[j]

- j不匹配，则j=next[j]
- j=-1则i++,j++
