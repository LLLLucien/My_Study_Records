# My Study Records

大三学年嵌入式方向学习笔记整理，涵盖 C 语言、Linux 系统、数据结构等课程内容。

## 目录结构

```
├── 大三上期/          # 大三上学期笔记
│   ├── C 语言基础     # 指针、数组、字符串、结构体、宏、动态内存等
│   ├── Linux 入门     # 操作系统概述、用户权限、进程管理、服务管理等
│   └── 项目实践       # 短网址服务等
│
└── 大三下期/          # 大三下学期笔记
    ├── 数据结构       # 线性表、栈、队列、树、图、排序算法等
    ├── Linux 文件系统  # IO操作、目录操作、线程、Base64编码等
    └── 阶段项目       # 电梯调度等项目文档
```

## 内容说明

- 笔记以 Markdown 格式记录，按日期命名
- 包含课堂代码示例、作业任务、知识点总结
- 仅收录 `.md` 笔记文件和 `.png` 截图

## 课程路线

`C 语言` → `Linux 操作系统` → `数据结构与算法` → `Linux 系统编程` → `嵌入式开发`
