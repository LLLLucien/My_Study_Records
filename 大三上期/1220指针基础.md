# 字符、字符串、二维数组

## 字符

char 字符类型（兼容整数类型）
字符：表示客观事物或抽象的概念的符号
举例：1、2、+、-、中、😀emji

字符编码    单字节字符/多字节字符
ANSII编码中最初只有128个
ANSII                     1字节    256
国标GBK/GB2312（简体）      2字节    65536
big5（繁体）
unicode                   1~4字节

拉丁字符       单字节   char
CJK（中日韩）  多字节   char ch[],char *ch,wchar_t.h(宽字符)

char从0~31都是不可打印出的东西，32~127都是常用的

小写的a-32即是大写的A
a（97）A（65）

```c
        // 判断是否是数字、字母
    char ch;
    printf("请输入一个字符：");
    scanf("%c", &ch);

    if (ch >= '0' && ch <= '9')
    {
        printf("这是数字\n");
    }

    // 也可以用正则表达式（模式匹配）
    if (ch >= 'a' && ch <= 'z' || ch >= 'A' && ch <= 'Z')
    {
        printf("这是字母\n");
    }
    
```

## 字符串

宽字符

```c
#include <stdio.h>
#include <wchar.h>
// i18n  Internationalization 国际化
// l10n  Localization 本地化
#include <locale.h> //本地化
int main()
{
    // 语言_地区.编码方式
    // CN指的是中国大陆，可以是zh_HK
    setlocale(LC_ALL, "zh_CN.UTF-8");
    // wide 宽字符：多字节字符
    wchar_t ch = L'王';
    // 用#，自动加一个0x
    wprintf(L"字符=%lc,编码值=%#x\n", ch, ch);
    return 0;
}
```

```c
    char s1[] = {'a', 'b', 'c'};
    //与下面的不等价

    // 字符串“”括住的文本内容，空间大小是元素大小加一

    char s3[] = {'a', 'b', 'c', '\0'};
```
