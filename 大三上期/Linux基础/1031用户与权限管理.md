# Linux用户管理和权限管理
- man 参考手册
- .conf配置信息文件
- .ls | grep 关键字（用来筛选）
- | 管道符号可以用来连接
- rm -f 文件（-f强制删除）
- rm -r 文件夹（递归）
- wc （统计文件多少行、单词、字节等）
- wc的参数-l（行）,-c（字节）,-m（字符）
- ls /etc > info.txt（把/etc的文件内容打印到info）
    - '>'（重定项，文件有内容会替换覆盖）
    - '>>'（不会覆盖，追加写）
- cat 文件名（打印文件内容）
- echo 文件名（重申）
- adduser aaa（添加用户）
- tail（显示文件最后的默认10行）
- tail -n 3 文件名（显示最后3行）
- head（同理，显示最前的默认10行）
- su 切换用户
- adduser aaa --gid 1004（指定添加到1004组）
- drwxr-wr-w
    - rwx（本人可以读、写、执行）
    - r-x（同组的可以读、不能写、执行）
    - r-x（其他人可以读、不能写、执行）
- deluser 删除用户（默认不删用户目录，里面还有文件）
-shadow（存储用户密码的地方）
- 文件类型
    - d 目录
    - ‘-’文件
    - link 快捷方式
    - ‘s’ 套接字
    - ‘c’ 字符设备
    - ‘b’ 块设备
- 权限
    - r 读
    - w 写
    - x 执行
    - ‘-’ 无权限
- 用户
    - ‘u’ 用户user
    - ‘g’ 同组group
    - ‘o’ 其他other
    - ‘a’ 所有人all
- 命令
    - 'chmod'修改权限
        - 例如chmod g+w b.txt
    - 'chown'修改所有者
        - 例如chown A b.txt
    - 'chgrp'修改组
- 用二进制快捷修改
    - rw- r-- r--
    - 110 100 100
    - 6---4---4
    - 例如chmod 644 b.txt
