### ssh
1. 安装服务器
    - sudo apt install openssh-server
2. 使用客户端连接：需要IP、用户名、密码
    - cmd
    - putty
    - Xshell
    - vscode
3. 查看是否安装了
    - service ssh status
    - systemctl status ssh
    - ps -ef | grep ssh（可通过进程查看，e表示所有，f表示详细）
