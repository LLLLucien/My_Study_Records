# 宏定义

- 无参数宏：常量
- 带参数宏：替代函数，速度快，但是源代码膨胀
- #的作用，转为字符串用“#”，拼接字符串用“##”

- 定义
- #define N 5
- 取消定义
- #undef N
- 如果没有被定义
- ifndef A

```c
    //如果没有被定义,则定义,避免重复include
    //my.h -> __MY_H
    #ifndef __MY_H
    #define __MY_H

    #endif 
```

```c
    #define PI 3.1415
    #define C(r) 2 * PI *(r) // 参数需要扩起来
    #define MUL(a, b) (a) * (b)
    #define MAX(a, b) (a) > (b) ? (a) : (b)

    // 函数重载
    // C中不可以用相同名，不同类型
    // 不过可以用宏

    //  参数不加括号，则是1+1*3+1
    //  参数加了括号，则（1+1）*（3+1）
    printf("%d\n", MUL(1 + 1, 3 + 1));
    printf("%d\n", MAX(789, 6));


    // 把a当字符串处理,宏可以嵌套
    #define STR(a) #a

    #define CAT(a, b) a##b
```
