# 组合数据结构

- 数组
- 结构体

## 结构体

```c
    //封装，自定义的类型
    struct Student{
        //姓名
        char name[16];
        //成绩
        int score;
        //性别
        char sex;
    }

    struct Student s1={"bob",89,'M'};
    struct Student s2={"alice",89};//可以缺失，但是得按顺序
    struct Student s3={.sex='F',.score=100,.name="jack"};//也可指定
    struct Student s4;

    s4.score=99;
    //名字是字符数组，不能直接赋值
    //需要这样
    strcpy(s4.name,"tom");

    //结构体用
    s1.score
    //而结构体指针用
    s1->score

    //用指针修改
    void update_score(struct Student *s, int n)
{
    s->score = n;
    // 等价
    (*s).score = n;
}
    

    //如果结构体里还有结构体，需要
    struct Student s1={"bob",89,'M',{2000,7,30}};
    //初始化可以这样做，但是以后不可以，只能这样
    s3.birthday.year=2000;
    //不可以
    s3.birthday={2000,7,30};
```

## 结构体的大小

内存对齐

- 最大类型长度的整数倍，1，2，4，8
- 建议从小到大
- 避免大小交叉存储

## 共用体/联合体 union

内存分布方式不同，内存是叠加的，共用内存，会覆盖

```c
    union U
    {
        char a;
        short b;
        int c;
    }
    union U u1;
    u1.a; //三个只能有一个存在
    u2.b;
    u3.c;
```
