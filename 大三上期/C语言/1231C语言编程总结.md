# 指针

## 概念

指针式一种数据类型，存储其他变量的空间地址，

### 数据类型

- 基本类型（原始类型）
- 组合类型（复合类型）：数组、结构体、联合体、枚举、类
- 指针
- void

## 定义

 类型 *变量名

 ```c
    int a;   //4
    char c;  //1
    double g;//8

    //指针变量，空间大小与平台有关 32位系统则是4字节，64位8字节
    int *p1;    //8
    char *p2;   //8
    double *p3; //8
```

```c
    char c;
    int i;
    double d;
    printf("%d\n",sizeof c);  //sizeof(运算符)

    p1 = &a;

    int r = *p1 //'*'叫做解引用或间接运算
    *p1 = 99;

    void *p;//可以存任何类型的，但是无法直接解引用
    p = &i;
    *p = 42;//不可以直接修改

    //需要进行类型强制转换
    int *px = (int *)p;
```

## 应用

- 函数的参数传递
- 减小内存浪费
- 数组操作（指针运算+,-）
