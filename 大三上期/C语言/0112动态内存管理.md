# 动态内存管理

- malloc
- free
- calloc 返回的空间被0填充
- calloc(1, 12);  单位大小+多少个单位，分配后用0填充

- realloc 扩容，分为原地扩容，或选另一个足够大的地方，把现有的搬过去
- ptr = realloc（ptr，size）需要接收一个参数，因为地址可能改变

- malloc 返回的是void*指针
//指定一个size字节大小的空间，返回一个指针用来指向该空间
- void *malloc(size_t size)

free()
所有被分配的空间需要手动释放
释放完后，指针会变为野指针

## void * （不是读空指针）

- 空指针是：NULL
- 野指针：没有初始化的，被free后的

- 通用指针
- 可以存储其他类型的指针的值
- 不能解引用，得转成具体的指针来解开

```c
    char c;
    int i;

    //具体指针
    char *p1=&c;
    int *p2=&i;

    void *p;//反正指针都是大小8
    //可以赋值给p，只是不同类型的需要操作的大小有区别
    //int只操作4个大小
    p=p1;
    p=p2;

    int *p3=(int *)p;//强制转换为int，现在可以操作4个大小的空间

```

## 流程

1.使用malloc分配内存,需要判断空指针if(p==NULL)

2.若空间不足，使用realloc重新分配，不过指针地址可能改变，需要接收参数
    ptr = realloc（ptr，size）

3.必须使用free 释放堆内存
    free(ptr);
    ptr=NULL;

4.calloc() 等价于 malloc() 加 memset(ptr,0,n);  
                        把n个地方的地址设置成0
