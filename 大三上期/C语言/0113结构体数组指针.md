# 结构体数组、指针

## 结构体

```c
//使用typedef
//这样可以创建变量时
//不需要再写struct Stu a1
//直接是Stu a1
typedef struct
{
    char name[16];
    int age;
} Stu;

Student a[3] = {{"吕布", 21}, {"赵子龙", 19}, {.name="貂蝉"}};
    //name是字符数组，初始化的时候可以直接‘=’，例如.name="貂蝉"
    //但之后只能用strcpy来赋值
    strcpy(a[3].name, "曹操");
    a[3].age = 34;

Student *p = a;
    for (int i = 0; i < (sizeof(a) / sizeof(a[0])); i++)
    {
        // printf("%s\t%d\n", a[i].name, a[i].age); //数组下标
        // printf("%s\t%d\n", (*p).name, (*p).age); //指针
        printf("%s\t%d\n", p->name, p->age); // 指针->
        p++;
    }
```
