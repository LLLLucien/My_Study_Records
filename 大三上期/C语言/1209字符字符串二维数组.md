## 数组

一块连续的存储空间，可以存储类型相同的多个元素，是一个容器
数组中元素有序存储，通过索引 index （下标）【随机】访问数组中的元素

避免：数组下标越界

```c
    //整型数组 a 容量是 5， 4byte * 5 
    int a[5];
    int[6]a;

    int n;
    n = 6;
    long b[n];
    n = 9; //后面改变n的值，不会再改变数组

    //二维数组
    int m [3][4];

    //数组初始化
    int a[6] = {100, 200, 300, 400, 500, 600};
    int a[6] = {0}//初始化第一个后，后面全都默认是0
    int a[]={1,2,3}//容量为实际大小

    //错误的初始化
    int a[3]
    a = {1,2,3};

    //声明之后初始化只能一个一个初始化

```
