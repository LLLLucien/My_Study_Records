# 运算 & 流程控制

## 流程控制

### 分支 - switch

```c
    //变量接收的类型：int,char,枚举
    switch (var)
    {
        case 1:
            //1
            break;
        case 2:
            //2
            break;
        case 'A';
            //3
            break;
        case 'Z':
        case 'z':
            //4
            break;
        default:
            //5
            //break;(可写可不写)
    }
```

### 循环结构

- `while`           次数不确定
- `do-while`
- `for`             使用循环计数器，循环次数确定

```c
    //先判断，再执行；循环体执行0~n次
    while()
    {
        //循环体
    }
    //先判断再执行；循环体执行1~n次
    do
    {
        //循环体
    }while();
    //1.循环变量初始化：1次
    //2.循环条件：1~n次
    //3.循环变量的改变【步长】
    //4.循环体：0~n次
    for(1;2;3)
    {
        //4
    }
```
