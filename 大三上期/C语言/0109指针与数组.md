# 字符串

字符串：字符的序列

字符串的实现：
    - 字符数组
    - 字符指针

```c
     //错误的字符串，只是字符数组
    char c1[]={'h','e','l','l','o'};
    char c1[]={'h','e','l','l','o','\0'}; //必须包含‘/0’
    
    char c1[]="hello";//最后隐藏了‘\0’
    char *c1="hello";

    char c1[5]="hello"//虽然看着是5个，但是需要n+1，因为还有个隐藏的‘\0’
    char c1[6]="hello"//才是对的

    //要打印用%s

    char s1[] = "hello";//不可修改整体
    s1="bye";//不可改变
    //但可以改变某个
    s1[0]='B';//可以改
```

## 区分字符数组与字符指针

```c
    char s1[] = "hello";//不可修改整体
    s1="bye";//不可改变
    //但可以改变某个
    s1[0]='B';//可以改

    char *s2 = "hello";//不可修改某个
    s2 = "bye";//可以改整体
    s2[0]='B';//不可以
```

## 函数

在标准库的string.h中定义了字符串相关的函数

- 普通版本的格式‘strxxx’：
- ‘strlen’查看字符串的长度
- ‘strcpy’拷贝
- ‘strcat’拼接
- ‘strcmp’比较 但是字符串比较不可用==
- ‘strchr’查找某一个字符
- ‘strstr’查找另一个字符串

- cat 可以查看文件，同时可以拼接文件内容
- cat a.txt b.txt > c.txt

-安全版本的格式‘strnxxx’：
多一个n，也就是需要操作的长度

- ‘strnlen’查看字符串的长度
- ‘strncpy’拷贝
- ‘strncat’拼接
- ‘strncmp’比较 但是字符串比较不可用==
- ‘strnchr’查找某一个字符
- ‘strnstr’查找另一个字符串

-‘memcpy’内存拷贝
-‘memmove’内存移动
-‘memset’内存填充（可以将特定内存区域修改全0）
-‘memcmp’内存比较

- scanf("%s", m);      // 读取到空格就不读了
- gets(m);             // 读取到换行就不读了
- fgets(m, 15, stdin); // 标准输入读取15个字符，写到m里面
- 用fgets，会读取回车，所以可以用
  - int len =strlen(m)-1;
    或者
  - int len = strlen(m);
  - m[len - 1]='\0'

```c

    char s1[] = "alice,jack,rose:tom";

        printf("%s\n", tok);
        // 1.参数NULL

        // 会修改原来的字符串，所以最好先复制备份
```
