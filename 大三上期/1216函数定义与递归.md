# 位运算 & 字符

## 位运算（效率高）

数值转化为二进制，逐位进行运算，结果数值

常见的运算符

- '&'   与
- '|'   或
- '^'   异或
- '~'   取反
- '>>'   右移
- '<<'  左移

### &

两个1得1，其他得0，掩码运算

```c
    int a = 2;      //0010
    int b = 3;      //0011

    //求一个数是奇数
    if(a & 1 == 1) //
    if(a % 2 == 1)
```

### | 或

有一个1就得到1，全0才0

通知

- 响铃   1  0001
- 震动   2  0010
- LED   4  0100

```c
    int a = 1;
    ina b = 2;
    a | b = 3;
    2 | 3 = 3;
```

### ^ 异或

一样则得0，不一样得1
相同的会消除
    1100
    1010
    ----
    0110

### ~ 按位取反

一元运算，按位取反

## >> 位移

<< 左移，在原数值的右侧补，乘以 2 的几次
>> 右移， 低位截断，左侧补0，除以 2 的n几次ff
