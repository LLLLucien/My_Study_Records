# 指针、数组、字符串

## 指针

指针是一种数据类型，存储其他变量的地址

```c
    //*,&
    int a=1;

    //* 声明指针类型变量
    int *p;

    //& 取地址
    p = &a;

    //* 解引用
    *p=3;
    int x = *p;

    //不可，类型不同，a为int型有4个字节地址
    //而char的指针只看一个字节
    char *p2=&a

    //野指针：声明后，未初始化或未赋值
    //动态内存分配（malloc）空间，释放后，也是也指针
    char *p2;
    *p2='A'; //不可，会出错

    //如何避免野指针
    //声明时进行初始化
    int *p = &a;
    int *p = NULL; //空指针


```

- 编译时可用
- gcc hello.c -Wall -o p
- 来显示警告

## 指针与数组

-

### 指针的运算（地址的偏移，单位为数据类型）

- 指针加法

```c
    char ch[9]={0};
    //ch是数组首地址
    for(int i = 0;i < 5;i++)
    {
        printf("%c",*(ch+i));
    }

    //不可以用
    ch++;
    ch = ch + 1;

    //但是可以
    char *p =ch;
    p++;

    while(p < ch + 5)
    {
        printf("%c\n",*p++)
    }
```

## 常量指针和指针常量

```c
    //常量指针:不可修改内容
    const int *p；

    //指针常量：不可指向其他地址
    int *const p;

    //指向常量的指针常量
    int const *const p;
```

## 指针函数

```c
    int* max(int *a,int *b)
    {
        int *r;
        if(*a>*b)
            r=a;
        else
            r=b;
        return r;
    }

    int *f()
    {
        //在函数外可以继续访问空间
        //必须手动释放free（p）
        int *p=malloc(sizeof(int));
        return *p;
    }

    // 指针函数
    int *f(int a, int b)
    {
        int r = a + b; // 局部变量

        return &r; // 返回了局部变量的地址，不可行
    }
    int main()
    {
        int *p = f(1, 2);
        printf("%d", *p);
        return 0;
    }
```
