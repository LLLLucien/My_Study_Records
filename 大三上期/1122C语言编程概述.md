# C语言编程

## 书籍：C专家编程

## 概述

### 语言的发展（标准）

- 1972 丹尼斯里奇基于B语言开发的UNIX，为了解决UNIX的操作问题，过程中设置了C语言，目标是简洁高效的控制硬件，实现系统编程
- 1978 出版《C Programming Language》（经典C）
- C89/90 ASNI C （第一个国际标准）
- C99：一些局部变量声明会不同
- C11 修复缺陷
- C17 增强一些并发编程相关的
- C23 新特性

### 编辑器

- GCC
- MSVC微软
- clang苹果
- tcc

### 编辑器 & IDE（集成开发环境）

- VS code
- vim
- dev C++
- x code

### 语言类型

静态类型：C、C++、Java
动态类型：Python、JS

### 编译型与解释型

编译型：C、C++、Java ->二进制代码文件
解释型：Python、JS、Java ->源代码直接跑

### C语言执行过程

- 源文件（.c）
- 预处理（预处理器，通常和编译合在一起）
- 编译：生成目标文件，在linux里是.o，在windows里是.obj
- 链接：目标文件、库文件、操作系统启动文件.exe
- 执行
