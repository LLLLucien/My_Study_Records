# 进程管理、网络管理
## Linux软件安装
- 包管理器（软件仓库）
    - apt（高级软件包管理工具）：主要是在debian、ubuntu 
    - yum：主要是在redHat、centOS、欧拉
- 应用市场（主要是GUI的）
- 安装文件
    - Windows的格式：exe、msi（微软安装包）
    - Linux的格式：deb（源自debian，）、rpm（全称redhat package manager）
    Linux压缩格式：.tar.gz
    window压缩格式：zip、rar（避免使用，此解压私有）
- 源代码（手工编译）
    - .tar.gz
    - .zip
- 如果安装过程中ctrl+z终止了，那么会有锁，解决方法：使用管理员身份删除rm 文件地址或者杀死某进程sudo kill 4475（某进程）
- whereis sl（可查看sl安装在哪里）
- ps 快照，查看这个窗口的所有进程
- top 实时显示进程
- htop 改良版
- ps -e是查看系统的所有进程
- ps -ef查看进程详细信息
- uname -r查看内核版本
-sudo kill 4544（可以加参数，例如sudo kill -9 4544）
- apt可以查看相关指令
    - apt install 文件名（安装）
        - 如果是自己下载的文件，不是从官方apt下载的，需要‘ ./ ’，从当前文件夹安装
    - apt remove 文件名（删除） 
    - apt update查看更新
    - apt upgrade执行更新
    - apt show 文件名（展示文件信息）
    - apt search 文件名（搜索）

### shell
一个软件程序，集成了各种命令程序，为用户提供一套一致的用户交互方式
- bash：系统默认
- fish：直接输入fish即可进入，这个比较方便，有代码补齐功能，使用向右键可以补齐，tab可以查看有哪些可以补齐的代码
- zsh：可以高度扩展可以自己定制，例如oh-my-zsh
- csh

## 打包
- .zip
- .tar.gz
- .rar
- c 创建压缩包
- v 显示详细信息
- f
### tar 
- 打包
    - tar -cvf abc.tar a.txt b.txt c.txt
- 压缩
    - tar -czvf abc.tar.gz a.txt b.txt c.txt
- 解压
    - tar -xzvf abc.tar.gz
### zip
- 压缩
    - zip -r abc.zip a p.txt
- 解压
    - unzip -x abc.zip
### rar
- 压缩
    - rar a abc.rar a p.txt
- 解压
    - unrar x abc.rar

### service
- ssh：openBSD secure shell
    - 服务器：openssh-server/sshd
    - 客户端：ssh

- service ssh status 显示状态
- service ssh start 启动服务
- service ssh stop 关闭服务
- service ssh restart 重启服务
