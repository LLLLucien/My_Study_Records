## 赋值

从右往左运算，右值：字面量、变量、表达式、函数
> 表达式：由操作数和运算符构成，2 + 3，a / 4
> 操作数：左操作数、右操作数，一个运算符加两个操作数叫二元运算，还有一元，三元等

```c
    int x = 1,y = 3,z = 9;
    x = y = z = 1;
    // 逗号运算：从左往右
    x = 2, 3;
    // 把两句合并成一条语句
    x = y + 1,y = 3;
```

## 算数运算

1. 模运算%

```c
    //模运算（只能是整数之间进行运算）
    printf("%d\n",7 / 4);   
    printf("%d\n",7 % 4);   // 取余3，而且如果是n%4,那么范围只能是0~3，可以用它来计数
    printf("%d\n",-7 % 4);  //负号在左边则结果为-3
    printf("%d\n",7 % -4);  //在右边则是3
    printf("%d\n",-7 % -4); //也是-3

    // 例如 n = 345
    //  模10除10法
    int n1, n10, n100;
    n1 = n % 10; // 5
    n /= 10;     // 34

    n10 = n % 10; // 4
    n /= 10;      // 3

    n100 = n % 10; // 3
    n /= 10;       // 0
```
