## 关系运算（比较运算）

- 常见运算符:
  - ==
  - !=
  - >
  - >=
  - <
  - <=
  
## 逻辑运算

- && 与，都真才真
  - 特点：短路，只要左边是假，另一个不会去看

- || 或，都假才假
  - 特点：短路，只要左边是真，另一个不会去看

- ！非，真变假，假变真

```c
    int age;
    //if(18 <= age <= 70)
    if(18<=age && age <70)
    
```

### 分支结构

- `if ()`
推荐用 if (a==7)写成if (7 == a)可以避免写成‘=’
- `if () else`
- `if () else if () else if() else`
- `switch(){case }`
- `三元运算 ? :`
- printf("%d 是 %s\n",n,n%2 == 0 ? "偶数":"奇数")

```c
    if () xxxx;
    //避免‘==’写成‘=’，非0的都算对，例如f（7）也算对
    
    if ()
        xxx;
    if ()
    {
        xxx;
        xxx;
    }
```
