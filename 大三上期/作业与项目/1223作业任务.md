# 函数

函数是构成C语言的基本单元
函数是命令的代码块、可复用的功能单元
函数是进程模块化开发的基础

编程范式：

- 脚本编程
- 函数：面向过程
- 类：面向对象

```c
#include <stdio.h>
// 当前路径的自定义头文件
#include "my.h"

// 声明、定义函数，参数：形式参数
// 模块化：高内聚、低耦合
// KISS原则：Keep It Simple Stupid 单一任务原则
// DRY原则：Don‘t Repeat Yourself
// 一个函数最好不超过100行

// 函数声明
// void print_array(int x[], int len);
int main()
{
    int a[] = {100, 200, 300, 400, 500};
    int b[20] = {0};
    // 函数调用

    print_array(a, 5); // x=a,,len=5
    print_array(b, 5); // a,b 实际参数
    

}

```
